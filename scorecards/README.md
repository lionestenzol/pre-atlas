# Scorecards · The Conversion Ladder

Daily scorecard framework adapted from Plan of Attack. Universal — works for sales, building, outreach, or any day where you move people/work through a funnel.

## Files

- `scorecard-template.md` — the blank template (source of truth)
- `scorecard-YYYY-MM-DD.md` — one per day
- `new-day.sh` — creates a fresh dated scorecard from the template

## Daily flow

**Morning:**
```bash
cd scorecards && ./new-day.sh
```
Open today's file. Fill in **X** (easy average) and **Y** (stretch).

**Throughout the day:**
- Tally the funnel (Attempts → Engaged → Qualified → Committed → Done)
- Rate the 6 behaviors after each work block (AM / MID / PM)

**Evening:**
- Compute conversion rates
- Use the diagnostic table: find the metric that dropped → fix the behavior behind it
- Fill the positive-only debrief (went well / proud of / do better / tomorrow's focus)

## The 5 stages

1. **Attempts** — every time you stepped into the arena
2. **Engaged** — they stopped, listened, looked up
3. **Qualified** — real fit, real interest, real need
4. **Committed** — verbal yes, scheduled, signed, handshake
5. **Done** — delivered, closed, paid, installed, shipped

## The 6 behaviors

Tempo · Presence · Opening Move · Hook · The Ask · Lightness

## Diagnostic principle

Metrics don't drop in isolation — each drop points back at a specific behavior. The debrief stays positive; the diagnostic table is where the honest adjustment lives.
