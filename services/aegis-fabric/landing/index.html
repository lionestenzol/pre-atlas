Declarative Policies

Define what agents can do with JSON rules. Match on agent, action, mode, time, or any context field. No code changes needed.

Human-in-the-Loop

Flag high-risk actions for human approval. Agents pause, humans decide, agents proceed. Configurable escalation per policy.

Audit Everything

Every action, every decision, every policy evaluation logged with full context. Immutable audit trail for compliance and debugging.

One API call. Full governance.

// Before your agent acts, check with Aegis
const decision = await fetch('https://your-aegis.com/api/agents/agent-1/actions', {
  method: 'POST',
  body: JSON.stringify({
    action: 'delete_user',
    params: { user_id: 'u_123' },
    context: { mode: 'production' }
  })
});

// Aegis returns: ALLOW, DENY, or REQUIRE_HUMAN
if (decision.effect === 'DENIED') {
  console.log(decision.reason); // "Policy 'no-delete-in-prod' denied this action"
}
