## Extracted Frameworks

### Quasi-Code Concept
A hybrid numeral system between binary (0-1) and decimal (0-9):
- Extends binary's simplicity with decimal's density
- Range-based encoding instead of strict positional notation
- Application: lossless file compression where quasi-code represents data more compactly than binary but more precisely than decimal
- Paired with vectorization for spatial/contextual relationships and indexing for O(1) lookups
