Think of it like this: Your conversations are like a messy room. The Brain Scanner goes through everything and makes a list of what's unfinished. The Rule Engine looks at that list and says "you have too many things open — you're in CLOSURE mode, finish stuff before starting new things." The Dashboard shows you that order on screen.

The 6 Behavior Modes

The system puts you in one of 6 modes based on your numbers. Each mode controls what you're ALLOWED to do. No AI picks this — it's pure math.

🛌 RECOVER
You slept less than 6 hours. Stop. Rest. Nothing else allowed.
→
🔒 CLOSURE
Too many open loops. Finish things. No new projects.
→
🔧 MAINTENANCE
Things are messy but manageable. Clean up, organize.
→
🏗️ BUILD
You're clear! Create new things, ship assets, go fast.
→
📈 COMPOUND
Building momentum. Stack wins on top of wins.
→
🚀 SCALE
Everything is clicking. Expand, delegate, multiply.
It's like a traffic light system: 🔴 RECOVER/CLOSURE = Red light, stop what you're doing. 🟡 MAINTENANCE = Yellow, slow down and fix things. 🟢 BUILD/COMPOUND/SCALE = Green light, go create.

The system won't let you run a green light when your numbers say red.

The 3 Main Services

Pre Atlas has 3 engines that work together. Each one has a specific job.

🧠
Cognitive Sensor
THE BRAIN SCANNER — reads your conversations
Python SQLite

This one reads ALL your past conversations (93,898 messages!) and figures out: What did you start but never finish? What topics keep coming up? How focused or scattered are you?

  • Open Loop Detector — Finds things you started but didn't finish
  • Closure Ratio — Calculates what % of things you actually complete
  • Mode Calculator — Decides if you should be building or closing
  • 8 AI Agents — Extract ideas, classify them, rank priorities
  • Cognitive Atlas — 84,000-point interactive brain map
  • Daily Governor — Generates your daily brief
  • Weekly Governor — Weekly review and planning
  • Book Miner — Extracts book-worthy content from your convos
⚙️
Delta Kernel
THE RULE ENGINE — enforces your behavior
TypeScript SQLite

This is the strict parent. It takes what the Brain Scanner found and applies hard rules. Every change is recorded forever (like a blockchain). No cheating, no faking progress.

  • Mode Router — The locked rulebook that picks your mode (no AI)
  • Governance Daemon — 6 automatic jobs running 24/7
  • Hash-Chain Deltas — Every change is permanent and tamper-proof
  • Closure Tracking — Records when you finish things + streaks
  • Work Queue — You must REQUEST permission to start new work
  • REST API — Port 3001, serves all dashboards
  • CLI App — Terminal interface for power users
  • Timeline Logger — Records every system event forever
🛡️
Aegis Fabric
THE SECURITY GUARD — controls AI agent access
TypeScript

This is the bouncer at the door. When any AI agent (Claude, OpenAI, etc.) wants to DO something, it has to ask Aegis first. Aegis checks the rules and says ALLOW, DENY, or ASK A HUMAN.

  • Policy Engine — Declarative rules (9 operators, 3 effects)
  • Agent Adapter — Normalizes Claude/OpenAI/custom formats
  • Approval Queue — Humans approve sensitive actions
  • Multi-Tenant — Supports FREE/STARTER/ENTERPRISE tiers
  • Audit Log — Records every agent action permanently
  • Webhook Events — Notifies external systems of actions
⚠️ Status: Built & compiled, but not yet connected to the main system

Bonus Features

Extra systems built on top of the core engine.

💡
Idea Intelligence
8 AI agents mine your conversations for ideas

Automatically extracts every idea you've ever mentioned, removes duplicates, scores them by priority, and ranks them into tiers: Execute Now, Next Up, Backlog, Archive.

  • Excavator — Finds raw ideas (18 regex patterns)
  • Deduplicator — Merges similar ideas
  • Classifier — Tags by category and status
  • Orchestrator — Ranks by 5-factor priority score
  • Reporter — Generates readable registry
📖
Book Pipeline
Turns your conversations into book chapters

Extracts the best passages from your conversations, ranks them by relevance, and drafts book chapters. Currently working on Chapter 1: "The Anatomy of Disrespect."

  • Book Miner agent — extracts raw material
  • Top 20 passages ranked by score
  • Draft → Final chapter pipeline
  • PDF builder (Markdown → PDF)
🗺️
Cognitive Atlas
Interactive 3D-style map of your brain

Takes 84,848 message embeddings, projects them into 2D space, clusters them, and builds an interactive visualization. You can literally SEE what your brain focuses on.

  • 6 view modes (Cluster, Role, Time, Convo, Leverage, Graph)
  • UMAP projection (384 dimensions → 2D)
  • HDBSCAN auto-clustering
  • ForceAtlas2 force-directed layout
  • Plotly WebGL + Sigma.js rendering

Known Issues

Things that work but could be better. Nothing is broken — these are optimization opportunities.

🔀
Triple Mode Routing
Three different places calculate your mode (Python, TypeScript LUT, TypeScript daemon). They could disagree. Should be one source of truth.
📋
Duplicated Code
The delta engine (delta.ts) is copy-pasted into both delta-kernel and aegis-fabric. If you change one, the other gets outdated.
🔓
No File Locking
Python and TypeScript both read/write the same JSON files without locks. They could step on each other (rare but possible).
🧪
Limited Tests
Only 7 Python tests (atlas pipeline). No TypeScript tests. No agent tests, no governance tests.
🔗
Aegis Not Connected
Aegis Fabric is fully built but not wired into the main system yet. It runs on port 3002 independently.
📂
Overlapping Pipelines
refresh.py, run_daily.py, and atlas_agent.py all do similar things. Should be one pipeline with options.

One Last Analogy

Pre Atlas is like a personal trainer for your brain:

🧠 Cognitive Sensor = The trainer who watches your workout (reads your conversations, finds what you're neglecting)

⚙️ Delta Kernel = The training program (strict rules: "you WILL do legs today because you skipped them 3 times")

🛡️ Aegis Fabric = The gym rules (any AI assistant that wants to help you train must follow the gym's policies first)

📺 Your Dashboards = The workout app on your phone (shows you today's exercises, your streaks, your progress)

The system doesn't care how you feel about it. If your numbers say CLOSURE mode, you're closing loops. Period. That's the whole point — it removes self-deception from your workflow.
